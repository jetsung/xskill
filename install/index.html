Quick install (Linux / macOS)

One-line install script — auto-detects OS and architecture:

curl -fsSL https://xskill.gcli.cn/install.sh | bash
Installs to ~/.local/bin (or /usr/local/bin for root).

Quick install (Windows)

PowerShell one-liner — auto-detects architecture:

irm https://xskill.gcli.cn/install.ps1 | iex
Installs to %USERPROFILE%\.local\bin (or %ProgramFiles%\xskill\bin for admin). The script will prompt you to add it to PATH if needed.

From crates.io

cargo install xskill

From Git

cargo install --git https://github.com/jetsung/xskill.git xskill
cargo install --git https://atomgit.com/jetsung/xskill.git xskill

From source

git clone https://github.com/jetsung/xskill.git cd xskill cargo install --path .
